# Portfolio | Tamara Korol

Навчальний сайт-портфоліо для лабораторних робіт з HTML, CSS, JavaScript та Git/GitHub.

## Склад проєкту

- `index.html` — головна сторінка портфоліо.
- `contacts.html` — сторінка контактів і форма з JavaScript-перевіркою.
- `style.css` — усі стилі сайту.
- `script.js` — клієнтські сценарії: тема, фільтри, сортування, модальне вікно, форма, localStorage.

## GitHub Pages

Після публікації сайт буде відкриватися за адресою:

`https://USERNAME.github.io/REPOSITORY-NAME/`

Заміни `USERNAME` на свій логін GitHub, а `REPOSITORY-NAME` — на назву репозиторію.

## Як запустити локально

Відкрий файл `index.html` у браузері.
